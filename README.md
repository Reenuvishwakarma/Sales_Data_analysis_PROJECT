# Sales Data Analysis (EDA Project)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on sales data to find useful business insights.

## Objective
The objective of this project is to analyze sales performance, identify top-performing products, understand regional sales trends, and study the relationship between sales and profit.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data loading and understanding
- Data cleaning
- Missing value checking
- Duplicate value checking
- Product-wise sales analysis
- Region-wise sales analysis
- Sales vs Profit relationship analysis
- Outlier detection using boxplot
- Correlation analysis using heatmap

## Key Insights
- Identified products generating maximum sales.
- Compared sales performance across different regions.
- Analyzed the relationship between sales and profit.
- Detected unusual values using boxplot analysis.
- Studied correlations between numerical features
